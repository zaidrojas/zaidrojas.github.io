Public vs Private Use

Aspects Public Use Private Use
Visibility Listed on the Chrome Web Store, searchable by anyone. Only visible to users you allow (via G Suite domain or direct install link). Not searchable publicly.
Audience Anyone with Chrome can install it. Limited: only your organization (if using Google Workspace) or individuals with the install link.
Use Case Sharing tools with the general public (ad blockers, productivity tools, themes, etc.). Internal tools for your company, school, or personal projects you don't want publicly accessible.
Updating Auto-updates for all users once you push a new version to the Web Store. Same update mechanism, but only affects your private audience.
Distribution Anyone can install directly from the Chrome Web Store listing page. Install via direct link, domain publishing, or local install in Developer Mode.
